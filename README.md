# files-downloader

> 常常有一批网址下载文件的时候，用迅雷或一些浏览器插件下载的文件目录无法保持一致且文件会因为冲突被重命名，所以为了方便自己开发了一小软件，觉得有用的来个star谢过！

### 功能
1. 根据文件目录自动生成文件目录
2. 支持大文件
3. 多任务同时下载
4. 可直接已下载打开文件/文件夹

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
